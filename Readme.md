# 🧠 Neural Ethics – Smart Resource Allocation System

## 📌 Overview
Neural Ethics is a smart web-based platform designed to connect NGOs and volunteers using an intelligent resource allocation system.

It ensures:
- Right Volunteer  
- Right Task  
- Right Time  

---

## 🎯 Problem Statement
NGOs and social initiatives lack an intelligent system to efficiently match volunteers with tasks based on:
- Skills  
- Availability  
- Location  
- Urgency  

This leads to:
- Poor resource utilization  
- Delays in critical tasks  
- Volunteer disengagement  

---

## 💡 Solution
The **Smart Volunteer Allocation System** solves this by:
- Matching volunteers using multi-factor logic
- Prioritizing urgent tasks
- Improving coordination between NGOs and volunteers

**Tagline:**
> Connecting the right volunteers to the right cause at the right time

---

## 🚀 Features

### 🔐 Authentication
- Login / Signup (NGO & Volunteer)
- Firebase Authentication
- Email verification

---

### 🧑‍🤝‍🧑 Role-Based Dashboards

#### 🏛️ NGO Dashboard
- Create / Edit / Delete tasks
- Add skills, location, urgency
- View assigned volunteers
- Track task progress

#### 🙋 Volunteer Dashboard
- Profile setup (skills, availability, location)
- View recommended tasks
- Accept / Reject tasks

---

### 🧠 Smart Matching System
- Skills-based matching
- Location-based filtering
- Availability check
- Urgency prioritization
- Match score display

---

### 🗺️ Google Maps Integration
- Task location selection
- Distance calculation
- Navigation support
- Live location tracking

---

### 📡 Live Tracking
- Real-time GPS tracking
- Firestore updates
- Map visualization

---

### 🔔 Notifications
- Push notifications (Firebase Cloud Messaging)
- Email alerts:
  - Login alert
  - Task assignment
  - New tasks available

---

### 🗄️ Database (Firestore)

Collections:
- users  
- tasks  
- assignments  
- notifications
